# Tony's Greenwich Village Pizzeria

A classic, no-frills, family-owned NYC pizzeria website with a modern web presence. This project captures the authentic, warm, and bustling atmosphere of a traditional Greenwich Village pizzeria, emphasizing high-quality ingredients and decades of tradition.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI/UX**: Clean, warm design with authentic NYC pizzeria vibes
- **Interactive Menu**: Tabbed menu interface for easy navigation
- **Smooth Scrolling**: Seamless navigation throughout the site
- **Mobile-Friendly**: Hamburger menu for mobile devices
- **Contact Form**: Easy way for customers to reach out

## Sections

1. **Hero Section**: Eye-catching landing area with call-to-action
2. **About**: Family story and tradition
3. **Menu**: Interactive menu with categories (Pizza, Calzones, Appetizers, Salads)
4. **Hours & Location**: Business hours and address
5. **Contact**: Contact form and social media links

## Color Scheme

- **Primary Red**: `#C41E3A` - Classic pizzeria red
- **Dark Red**: `#8B1538` - Deep, rich red
- **Warm Orange**: `#FF6B35` - Accent color for CTAs
- **Cream**: `#F5F1E8` - Warm background tones
- **Charcoal**: `#2C2C2C` - Text and dark elements

## Typography

- **Headings**: Playfair Display (elegant serif)
- **Body**: Lato (clean, readable sans-serif)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process or dependencies required - pure HTML, CSS, and JavaScript

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

### Adding Images

Replace the placeholder divs with actual images:
- Hero section background
- About section family photo
- Map location

### Updating Content

- Edit text directly in `index.html`
- Modify menu items in the menu sections
- Update hours and contact information

### Styling

- All styles are in `styles.css`
- Color variables are defined in `:root` for easy customization
- Responsive breakpoints are set at 768px and 480px
